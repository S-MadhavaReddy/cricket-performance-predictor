Cricket Performance Predictor

This project predicts:

• Runs for batsmen
• Wickets for bowlers

Model Used:
Random Forest Regressor

Features Used:

Batting
- Matches
- Innings
- Strike Rate
- Hundreds
- Fifties
- Ducks

Bowling
- Matches
- Innings
- Balls
- Maidens
- Runs
- Economy
- 5W

Deployment:
Streamlit
